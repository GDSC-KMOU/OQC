## 흐름

create부터 text까지 SPA

메인화면 -> 회원가입(로그인) -> 폐기 신청(create) -> 사진 업로드 -> 사진 반환 결과 토대로 클라이언트가 사이즈 선택 -> 사이즈에 대한 값 확인 후 폐기물 언제 어디에 버릴지 신청 등록 -> 신청 완료(text) -> 결제 -> 결제 완료 시 업체가 승인 -> 끝

사용 페이지: 메인화면 / 회원가입 / 로그인 / 모든 신청 글 보기 / 단일 신청 글 보기 / 신청 / 개인 신청 확인
<br><br>
